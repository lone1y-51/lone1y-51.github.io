# 位运算

`abs(x) = (x ^ (x >> 63)) - (x >> 63)` 针对64位的整数，8位或者32位的同理，将63改为7或者31

原理：以64位为例

正数 `>>63`得到`0x00` 负数得到`0xFF`

对于正数来说，上述运算等于`x^0-0`等于原值

对于负数来说，上述运算等于`x^0xFF-0xFF`等于取反+1，负数的取反加一（也就是补码）等于他的绝对值
